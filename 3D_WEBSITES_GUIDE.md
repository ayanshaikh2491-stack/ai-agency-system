## 🔧 HOW IT WORKS

### Stage 1: Code Generation (50ms)

```javascript
// Template-based instant generation
const files = await generateWebsiteFiles3D(config, paletteIndex);
```

### Stage 2: File Write (100ms)

```javascript
// Parallel disk writes
await writeFilesParallel(projectDir, files);
```

### Stage 3: Git Push (1-2 seconds, parallel with Vercel)

```bash
git init
git add .
git commit -m "Initial commit"
git push origin main
```

### Stage 4: Vercel Deploy (2-3 seconds, parallel with Git)

```bash
vercel --prod --token=xxx
```

**Key optimization:** Git and Vercel happen simultaneously!

## 💡 CUSTOMIZATION

### Change Products

Edit `pipeline/fastCodeGenerator3D.js`:

```javascript
const PRODUCT_DATA = {
  gym: [
    { name: 'Dumbbell Set', price: '$49.99', image: '🏋️' },
    // Add more...
  ],
};
```

### Change Colors

Edit `pipeline/fastCodeGenerator3D.js`:

```javascript
const COLOR_PALETTES = [
  {
    primary: '#1a1a2e',
    accent: '#0f3460',
    highlight: '#e94560',
    light: '#16a085',
  },
  // Add more...
];
```

### Add Business Types

1. Add config to `WEBSITE_CONFIGS_3D`
2. Add color palette to `COLOR_PALETTES`
3. Add products to `PRODUCT_DATA`
4. Run deployer

## 🚨 TROUBLESHOOTING

### "GITHUB_TOKEN not found"

```bash
export GITHUB_TOKEN="ghp_xxxxx"
echo "Check: echo $GITHUB_TOKEN"
```

### "VERCEL_TOKEN not found"

```bash
export VERCEL_TOKEN="vercel_xxxxx"
echo "Check: echo $VERCEL_TOKEN"
```

### Git push timeout (⚠️ warning message)

```
⚠️ Git slow, continuing...
```

This is OK! Site still deploys. Git might have been slow. Check Vercel dashboard.

### Vercel deploy exceeds 10 seconds

This is normal for first deploy. Subsequent deployments are faster.

### Website doesn't respond

Wait 30 seconds for Vercel to fully deploy. Then refresh.
